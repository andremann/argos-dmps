# Argos DMP Fetcher & Downloader

A Python script to bulk-fetch and download Data Management Plans (DMPs) from [OpenAIRE Argos](https://argos.openaire.eu), which is built on [OpenCDMP](https://opencdmp.eu).

The tool runs in two sequential steps:
1. **Fetch IDs** — queries the public Argos API to collect the IDs of all DMPs (latest versions only) and saves them to a file.
2. **Download XMLs** — uses those IDs to download each DMP as an XML export, one file per DMP.

---

## Requirements

- Python 3.7+
- [`requests`](https://pypi.org/project/requests/) library

Install the dependency with:

```bash
pip install requests
```

## Files

| File | Description |
|------|-------------|
| `argos_fetch_and_download.py` | Main script |
| `token.txt` | Your Bearer token (create this before running) |
| `dmp_ids.txt` | Generated by the script — list of DMP IDs, one per line |
| `argos_xml_exports/` | Generated by the script — folder containing downloaded XML files |

## Setup

### 1. Get a Bearer token

The XML export endpoint requires authentication. To get your token:

1. Log in at [https://argos.openaire.eu](https://argos.openaire.eu)
2. Open your browser's Developer Tools → Inspector and head to the **Network** tab
3. Perform any action (e.g., refresh or open a DMP)
4. Find a request to `argos.openaire.eu/api/...` and copy the `Authorization` header value
5. Strip the `Bearer ` prefix — keep only the token string itself

### 2. Save the token

Paste your Bearer token into the file called `token.txt` in the same directory as the script.

> ⚠️ Do **not** include the `Bearer ` prefix — just the raw token string.

---

## Usage

```bash
python fetch_dmps.py
```

The script will:

1. Query the Argos public API and collect all DMP IDs
2. Save the IDs to `dmp_ids.txt`
3. Download each DMP as an XML file into the `argos_xml_exports/` folder
4. Skip any DMP that has already been downloaded (safe to re-run)

> ⚠️ Uncomment the line with `versionStatuses` if you want only the latest versions of DMPs.


---

## Token Expiry

Bearer tokens expire after a short period. When the script hits a `401` or `403` response, it will **pause automatically** and print instructions.

Simply overwrite `token.txt` with the new token — **no need to restart the script**. It polls `token.txt` every 5 seconds and resumes from where it left off as soon as a new token is detected.

## Configuration

All settings are at the top of the script:

| Variable | Default | Description |
|----------|---------|-------------|
| `QUERY_URL` | Argos public query endpoint | API endpoint for fetching IDs |
| `EXPORT_URL` | Argos XML export endpoint | API endpoint for downloading XMLs |
| `IDS_FILE` | `dmp_ids.txt` | Output file for DMP IDs |
| `OUTPUT_DIR` | `argos_xml_exports` | Output folder for XML files |
| `TOKEN_FILE` | `token.txt` | File to read/refresh the Bearer token from |
| `PAGE_SIZE` | `1000` | Number of IDs fetched per API page |

---

## Notes

- The script adds a 0.1s delay between downloads to avoid hammering the server.
- On network errors, the script waits 5 seconds and retries the same DMP automatically.
- The ID fetch step (Step 1) does **not** require authentication — it uses the public API.
